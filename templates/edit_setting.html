message format has a URL like syntax.
  • say: takes two additional arguments, alice:en-GB or alice:en-US to indicate the language option.
  • play: is followed by a URL that points to a wave or mp3 file. It is important that the file format is wave or mp3 any other format and calls will drop.

example:

say:alice:en-GB:thank you for calling this call will be recorded.
